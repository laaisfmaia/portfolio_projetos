    Projetos em Ciência de Dados

    Identificação de imóveis para compra e revenda a fim de maximizar o lucro

    Objetivo:

    Identificação de imóveis abaixo do preço médio de venda e definição do preço ideal de revenda, a partir de uma análise exploratória de dados em Python.

    O objetivo é responder as perguntas de negócio:

  • Quais são os imóveis devem ser comprados e por qual preço?
  • Uma vez o imóvel comprado, qual o melhor momento para vendê-lo e por qual preço?
  • Quando ele deve ser vendido para ter a maximização do lucro?

    • Ferramentas utilizadas:

    • Python, Pandas, Matplotlib, Numpy e Seaborn;
    • Anaconda, Pycharm e Jupyter Notebook;
    • Mapas interativos com Plotly e Folium;
    • Heroku Cloud;
    • Streamlit Python framework web.

    Resultados financeiros:

    A análise permitiu encontrar 9 casas que estão aptas para a compra. A utilização dessa metodologia no auxílio na tomada de decisão para definição do preço de compra e revenda dos imóveis do portfólio proporcionará um lucro de aproximadamente U$2bi.

    Previsão de vendas de um conjunto de lojas farmacêuticas

    Objetivo:

    Esse projeto tem como objetivo principal prever o faturamento das próximas 6 semanas de algumas lojas para que o CFO consiga usar parte desse faturamento para a reforma dessas lojas.

    A motivação é a dificuldade em determinar o valor que pode ser investido para a reforma de cada loja.

    Ferramentas utilizadas:

    • Python: Pandas, Matplotlib, Numpy, Seaborn, Sklearn e Scipy;
    • Anaconda e Jupyter Notebook;
    • Git e Githut;
    • Render Cloud; Flask e Python API's;
    • Algoritmos de Classificação e Regressão: como Linear Regressor, Linear Regressor Regularized, Random Forest e XGBoost Classifier.

    Conclusão:

    O objetivo do projeto de prever o faturamento das lojas nas próximas 6 semanas foi atingido, resolvendo o problema de determinar o valor que pode ser investido para a reforma de cada loja.

    O algoritmo usado para a predição foi o XGBoost Regressor que apresentou um erro absoluto médio (MAE) de $761 (± 11%).
